An interactive Power BI dashboard analyzing health and lifestyle data to uncover trends in wellbeing, habits, and risk factors.**

This project transforms raw health & lifestyle survey/dataset data into a clean, interactive dashboard — enabling quick, visual exploration of patterns across demographics, habits, and health outcomes.

---

## ✨ Features

- 📈 **Interactive Visuals** — Drill-down charts, slicers, and filters for dynamic data exploration
- 🧍 **Demographic Breakdown** — Analysis by age, gender, and lifestyle segments
- 🥗 **Lifestyle Insights** — Correlations between diet, exercise, sleep, and health metrics
- ⚠️ **Risk Indicators** — Visual flags for key health risk factors across the dataset
- 🎯 **KPI Cards** — At-a-glance summary metrics for quick decision-making

---

## 🛠️ Tech Stack

| Component | Tool |
