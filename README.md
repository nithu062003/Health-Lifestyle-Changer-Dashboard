# Health-Lifestyle-Changer-Dashboard
Developed an interactive Power BI analytics dashboard to track 7+ health KPIs, facilitating trend analysis, behavioral monitoring, and data-driven wellness insights.
Designed advanced data models and custom DAX measures, along with automated reporting workflows to enhance
analytics, benchmarking, and performance monitoring.
Built an end-to-end data transformation and visualization pipeline using SQL, Power Query, and Power BI, converting raw
health data into actionable business intelligence.
Technologies / Tools Used : Python, SQL, Excel, Power BI, DAX, Power Query
